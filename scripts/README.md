# Scripts Directory

Utility scripts for OSPF6 analysis and debugging.

## Scripts

### `analyze_ospf6_spf.sh`
Analyzes OSPF6 SPF calculation logs.

**Usage**:
```bash
./scripts/analyze_ospf6_spf.sh
```

### `apply_ospf6_debug.sh`
Applies OSPF6 debug logging to running containers.

**Usage**:
```bash
./scripts/apply_ospf6_debug.sh
```
